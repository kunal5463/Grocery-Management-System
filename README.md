# Grocery-Management-System
Full Stack Grocery Management System using Node.js, Express, MySQL and modern UI

Technologies used:

Frontend:
HTML
CSS
JavaScript

Backend:
Node.js
Express.js

Database:
MySQL

Features:

User login and registration
Product listing page
Add to cart system
Update quantity
Remove items from cart
Place order
Order history page
Admin can manage order data

This project demonstrates real ecommerce workflow.
